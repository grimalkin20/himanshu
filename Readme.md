## How to Use

### HTML Resume
1. Open `resume.html` in any web browser
2. To print or save as PDF: Press Ctrl+P (Windows) or Cmd+P (Mac) and select "Save as PDF"

### DOCX Resume
1. Open `Himanshu_Kumar_Resume.docx` in Microsoft Word
2. Edit and customize as needed
3. Save or export to PDF

## Customization

### To Update HTML Resume:
- Edit `resume.html` for content changes
- Edit `resume.css` for styling changes

### To Regenerate DOCX:
```bash
python generate_resume_docx.py
```
